# Doom & Deeds

A simple local 2-4 player, cute-spooky property party game.

Every turn has one choice: roll. The game automatically moves the token, buys
free land, pays rent, and handles rewards or hazards. After eight rounds, the
player with the most coins and land wins.

## Run

```bash
npm install
npm run dev
```

## Checks

```bash
npm test
npm run build
```
